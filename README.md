#Flappy Bird Python
Um jogo simplificado inspirado no Flappy Bird, feito com Pygame.

##Como jogar
- Aperte **espaço** para pular.
- Evite os canos verdes.
- Cada cano passado aumenta sua pontuação.

##Tecnologias
- Python
- Pygame
